# 🔐 ManagerPasswords - Менеджер паролей

**Безопасное хранилище ваших учетных данных с шифрованием и удобным интерфейсом**

---

## 📋 О проекте

ManagerPasswords — это десктопное приложение для безопасного хранения и управления паролями. Приложение использует современное шифрование для защиты ваших данных и предоставляет интуитивно понятный интерфейс для работы с учетными записями.

### ✨ Основные возможности

- 🔒 **Шифрование данных** - все пароли шифруются алгоритмом Fernet (AES)
- 👥 **Многопользовательский режим** - разделение на роли (Администратор/Пользователь)
- 🎲 **Генератор паролей** - создание надежных паролей заданной длины
- 📋 **Быстрое копирование** - копирование паролей в буфер обмена в один клик
- 👁️ **Показ/скрытие паролей** - возможность просмотра сохраненных паролей
- 📊 **Журналирование** - отслеживание всех действий в системе
- 🎨 **Современный UI** - темная тема на базе CustomTkinter

---

##  Технологии

- **Язык:** Python 3.8+
- **GUI:** CustomTkinter
- **База данных:** SQLite
- **Безопасность:** 
  - bcrypt (хеширование паролей)
  - cryptography (шифрование данных)
- **Дополнительно:** pyperclip (работа с буфером обмена)
